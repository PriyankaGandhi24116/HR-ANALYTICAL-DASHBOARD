# 📊 HR Analytical Dashboard

## 📌 Overview

The **HR Analytical Dashboard** is an interactive Power BI project that transforms HR data into actionable insights. It helps HR professionals and business leaders monitor employee demographics, workforce distribution, job roles, education levels, and other key HR metrics through interactive visualizations, enabling informed, data-driven decision-making.

---

## 🚀 Features

- Employee Workforce Analysis
- Department-wise Employee Distribution
- Gender Distribution
- Education-wise Employee Analysis
- Job Role Analysis
- Average Employee Age
- Interactive KPI Cards
- Dynamic Filters & Slicers
- Data-Driven Visualizations

---

## 🛠️ Tools & Technologies

- **Power BI Desktop**
- **Microsoft Excel**
- **Power Query**
- **DAX (Data Analysis Expressions)**
- **Data Modeling**
- **Data Visualization**

---

## 📊 Dashboard Insights

This dashboard helps answer key HR questions such as:

- What is the total number of employees?
- How are employees distributed across departments?
- What is the average employee age?
- What is the gender distribution of employees?
- How are employees distributed by education level?
- Which job roles have the highest employee count?
- What workforce trends require management attention?
